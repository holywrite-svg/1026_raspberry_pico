##使用UV
-在樹莓派（Raspberry Pi）上安裝 uv（即 Astral 的超快速 Python 套件管理工具）其實非常簡單，只要幾個指令就能完成。以下是詳細步驟 👇
-🧰 一、確認你的環境
-首先請確定你有：
-Raspberry Pi OS（或其他基於 Debian 的 Linux）
-已安裝 curl（通常預設就有）
-可先執行：
- sudo apt update
sudo apt install curl -y
🚀 二、安裝 uv
官方推薦使用這個安裝指令（適用所有 Linux）：
curl -LsSf https://astral.sh/uv/install.sh | sh
這會自動：
下載適合你 CPU 架構（ARM、ARM64、x86）的 uv 二進位檔
安裝到 ~/.local/bin/uv
⚙️ 三、將 uv 加入 PATH（若還沒自動設定）
執行以下命令讓系統能找到 uv：
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
🧪 四、確認安裝成功
輸入：
uv --version
如果能看到版本號（例如 uv 0.5.x），就代表成功了 🎉
📦 五、用法範例
安裝套件（類似 pip）：
uv add requests
建立新虛擬環境：
uv venv
或直接執行 Python：
uv run python app.py
如果你用的是特別舊的樹莓派（例如 Raspberry Pi Zero 或 3B），可能會遇到架構不支援的情況（特別是 32-bit OS）。
那時可以：
改裝 64-bit Raspberry Pi OS
或用 pipx 安裝 uv（會稍慢但相容）：
pip install --user uv

## 更完整的 Python 函式庫清單

除了先前列出的常用函式，Python 還提供了更多強大的工具。以下是更詳盡的清單，但請注意這並非全部，完整的列表需參考官方文件。

### 更完整的 Python 內建函式 (Built-in Functions)

#### 數學與數值運算
*   **`abs()`**: 回傳一個數的絕對值。
*   **`round()`**: 將數字四捨五入到指定的小數位數。
*   **`pow(x, y)`**: 計算 x 的 y 次方，等同於 `x ** y`。

#### 序列與迭代處理 (Iterables)
*   **`enumerate()`**: 在迴圈中同時取得索引 (index) 和值 (value)。
*   **`zip()`**: 將多個序列的元素「打包」在一起，方便平行處理。
*   **`map()`**: 將一個函式應用於序列中的每一個元素。
*   **`filter()`**: 根據一個函式過濾序列中的元素。
*   **`all()`**: 判斷一個序列中的所有元素是否都為 `True`。
*   **`any()`**: 判斷一個序列中是否有任何一個元素為 `True`。

#### 物件與類別 (Object & Class)
*   **`isinstance()`**: 檢查一個物件是否為某個類別的實例。
*   **`hasattr()`**: 檢查一個物件是否有指定的屬性。
*   **`getattr()`**: 取得一個物件的屬性值。
*   **`setattr()`**: 設定一個物件的屬性值。
*   **`dir()`**: 列出一個物件所有的屬性和方法。

#### 檔案與程式互動
*   **`open()`**: 開啟一個檔案，並回傳檔案物件，是檔案處理的基礎。
*   **`help()`**: 顯示物件的說明文件。

### 更多常用的標準函式庫 (Standard Library)

這些是 Python 內建的強力模組，能處理更複雜的任務。

#### 系統與檔案操作
*   **`sys`**: 提供存取由 Python 直譯器使用的變數和函式，例如命令列參數 (`sys.argv`)。
*   **`pathlib`**: 以物件導向的方式來操作檔案系統路徑，是 `os.path` 的現代化替代方案。
*   **`shutil`**: 提供高階的檔案操作，例如複製、移動、刪除整個目錄樹。
*   **`subprocess`**: 用於執行外部指令或程式。

#### 資料格式與處理
*   **`re`**: 提供正規表示式 (Regular Expression) 的功能，用於強大的字串匹配與處理。
*   **`csv`**: 用於讀寫 CSV (逗號分隔值) 格式的檔案。
*   **`collections`**: 提供額外的高效能容器資料型別，例如 `Counter` (計數器)、`defaultdict` (預設值的字典) 等。

#### 網路與 Web
*   **`urllib`**: 用於處理 URL 和發送 HTTP 請求的模組集合。
*   **`http.server`**: 可以快速啟動一個簡單的本機 Web 伺服器。
*   **`socket`**: 提供底層的網路通訊介面。

#### 其他實用工具
*   **`itertools`**: 提供一系列用於高效迭代的工具函式。
*   **`functools`**: 提供高階函式和可呼叫物件的操作。
*   **`logging`**: Python 內建的日誌系統，比 `print()` 更靈活、更強大。
*   **`sqlite3`**: 一個輕量級的磁碟資料庫，不需要獨立的伺服器進程。

## 再補充一些實用的內建函式

這裡再補充幾個在特定情境下非常實用的內建函式。

### 字元與編碼

*   **`ord()`**: 將一個字元轉換為其對應的 Unicode 編碼（整數）。
*   **`chr()`**: 將一個 Unicode 編碼（整數）轉換為其對應的字元。

    ```python
    # 字元轉編碼
    unicode_code = ord('A')
    print(f"'A' 的 Unicode 編碼是: {unicode_code}") # 輸出 65

    # 編碼轉字元
    character = chr(65)
    print(f"編碼 65 對應的字元是: {character}") # 輸出 'A'
    ```

### 物件內省 (Introspection)

*   **`id()`**: 回傳一個物件在記憶體中的唯一識別碼。可以用來判斷兩個變數是否指向同一個物件。
*   **`callable()`**: 判斷一個物件是否可以被「呼叫」（例如函式或帶有 `__call__` 方法的物件）。
*   **`repr()`**: 回傳一個物件的「官方」字串表示形式，目標是讓這個字串能被 `eval()` 用來重建物件。

    ```python
    def my_function():
        pass

    x = 10

    print(f"函式是可呼叫的嗎? {callable(my_function)}") # 輸出 True
    print(f"變數 x 是可呼叫的嗎? {callable(x)}")       # 輸出 False
    print(f"x 的官方表示法: {repr(x)}")               # 輸出 '10'
    ```
