## 所需元件

*   Raspberry Pi Pico W (已焊接排針) [15]
*   麵包板 [8]
*   10KΩ 可變電阻 [8, 10]
*   LED (任意顏色)
*   按鈕 (Tactile Switch)
*   220Ω 電阻
*   杜邦線 (公對公) [8]
*   Micro USB 傳輸線

## 接線說明

請依照以下步驟將可變電阻連接到 Pico W：

1.  **斷開 Pico W 的電源**：在接線前，請務必將 Pico W 與電腦的 USB 連接斷開。
2.  **將 Pico W 固定在麵包板上**：將 Pico W 插入麵包板的中央凹槽，確保兩側接腳不互相接觸。
3.  **LED**: `GP15` → 220Ω 電阻 → LED 長腳(+) → LED 短腳(-) → `GND`
4.  **按鈕**: `GP14` → 按鈕一腳, 按鈕另一腳 → `GND`
5.  **可變電阻**:
    *   外側腳 1 → `3V3(OUT)`
    *   中間腳 → `GP28` (ADC2)
    *   外側腳 2 → `GND`
